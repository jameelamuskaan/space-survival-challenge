# Space Survival Challenge

A simple Scratch game where the player controls a spaceship and tries to survive in space by avoiding obstacles.

## How to play

- Open the file `Space Survival Challenge (1) .sb3` in Scratch 3.0 or on the Scratch website.
- Use the arrow keys to move the spaceship.
- Avoid obstacles and survive as long as possible.
